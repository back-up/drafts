## Babel Plugin

## Babel

Babel 共分为 3 个阶段
* parse 解析
* transform 转换
* generate 生成

## Parse

Parse 解析阶段分为
* Lexical Analysis 词法分析，将字符切分为 tokens
* Syntactic Analysis 语法分析，将 tokens 转换为 AST 结构
